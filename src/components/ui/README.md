# src/components/ui

Generic shadcn/ui primitives generated by the project scaffold.

Only `sonner.tsx` (the toast notification component used by the Contact
form's submit feedback) is actually imported anywhere in this codebase —
verified by grepping every `.tsx`/`.ts` file for `components/ui` imports.

The other ~40 shadcn primitives that ship by default (button, dialog,
card, table, accordion, carousel, etc.) were scaffolded but never wired
up to anything, so they were removed as dead code during the project
reorg. `components.json` (at the repo root) still has the shadcn CLI
config, so if you need one of them later:

```
npx shadcn add button
```
